title: 常用快捷键
slug: /tools/IDE/jetbrains/common-shortcut-key
sidebar_position: 2
---

- 前进：`Command` + `Option` + `←`
- 后退：`Command` + `Option` + `→`
- 切换上一个窗口：
  - `Command` + `Shift` + 反引号`
  - `Shift` + `Command` + `[`
- 切换下一个窗口：
  - `Command` + 反引号`
  - `Shift` + `Command` + `]`
  - `Command` + `~`
- 显示最近打开的文件：`Command` + `E`
